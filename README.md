# 🚀 Interns Portal - MERN Stack Internship Management System

A full-stack web application built to connect **students, companies, and admin** for managing internship opportunities, applications, and verifications.
This project is part of a Final Year Project (FYP) and follows a real-world SDLC approach from requirement analysis to deployment.

# 📌 Features

## 👨‍🎓 Student Panel
- Student registration & login
- Browse internships
- Apply for internships
- Upload resume
- Track application status (Pending / Accepted / Rejected)
- Profile management

---

## 🏢 Company Panel
- Company registration & login
- Post internships
- Manage internship listings
- View student applications
- Approve / reject candidates
- Company profile management

---

## 🛡 Admin Panel
- Verify companies (document approval system)
- Verify students
- Manage platform users
- View system analytics
- Monitor internship activities

---

## 🔔 Notification System
- Application submitted alerts
- Application status updates
- Company verification status
- Admin notifications for new registrations

---

## 📅 Internship Features
- Internship posting with deadline (Last date to apply)
- Auto-close applications after deadline
- Real-time status updates

---
